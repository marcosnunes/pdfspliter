Instruções

1. Clique em "Escolher Arquivos PDF" para selecionar os arquivos PDF que você deseja unir.
2. A ordem dos arquivos selecionados será a ordem em que serão unidos no PDF final.
3. Clique no botão "Unir PDFs".
4. Um link para download do PDF unificado será gerado após o processamento.
