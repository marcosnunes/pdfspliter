Instruções

1. Clique em "Selecionar PDF" para escolher um ou mais arquivos do seu computador.
2. O nome do arquivo selecionado aparecerá abaixo do botão.
3. Clique no botão "Processar PDF".
4. Aguarde enquanto o sistema divide o PDF em páginas individuais e tenta identificar o nome do prestador em cada uma.
5. As páginas processadas aparecerão na seção "Páginas Processadas" abaixo, com botões para download. Cada arquivo será nomeado com o nome do prestador (se encontrado).
